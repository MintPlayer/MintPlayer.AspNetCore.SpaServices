This package contains abstractions and interfaces for `MintPlayer.AspNetCore.SpaServices`. It is typically referenced by libraries that need to work with SPA services without taking a direct dependency on the implementation.

## Installation

### NuGet Package Manager
```
Install-Package MintPlayer.AspNetCore.SpaServices.Abstractions
```

### .NET CLI
```
dotnet add package MintPlayer.AspNetCore.SpaServices.Abstractions
```

## Usage

This package is primarily used as a dependency by other packages in the MintPlayer.AspNetCore.SpaServices ecosystem. You typically don't need to reference it directly unless you're building a library that integrates with SPA services.
